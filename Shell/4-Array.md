# Array
여러 값을 가지는 경우 배열을 이용하게 된다.
## 배열 초기화
배열을 선언하는 방법은 아래와 같다.

```bash
arr=()
```
배열 초기화 시 값을 넣고 싶다면 ``=``오른쪽에 대입할 값을 작성하면 된다.
** 스페이스 공간을 이용해 구분한다.
```bash
arr=("ex1" "ex2")
```
## 배열 조회
배열 전체를 출력할 때는 ``@``를 이용한다.
```bash
arr=("ex1" "ex2")
echo ${arr[@]}
```
배열의 특정 순서의 값을 출력할 때는 해당 인덱스값을 ``[]``사이에 위치하여 출력한다.(인덱스는 0부터 시작)
```bash
arr=("ex1" "ex2")
echo ${arr[1]}
```
배열 반복은 다음과 같다.
```bash
arr=("ex1" "ex2")
for i in ${arr[@]}
do
  echo ${arr[i]}
done
```
배열의 크기를 출력할 때는 다음과 같다.
```bash
arr=("ex1" "ex2")
echo ${#arr[@]}
```
